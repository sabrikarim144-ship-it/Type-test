🚀 Ma Page de Révision : Tests Logiciels

Niveau 1
Test Unitaire (TU)

On teste une fonction ou une méthode de manière isolée. Outil : JUnit (Java), PyTest (Python).

Niveau 2
Test d'Intégration (TI)

On vérifie que les modules s'emboîtent bien. On teste les échanges de données entre classes ou services.

Niveau 3
Test d'Intégration Système (SIT)

On vérifie que notre application communique bien avec des systèmes externes (APIs tiers, bases de données distantes).

Niveau 4
Test Système

Test complet "bout-en-bout" (E2E). On vérifie si la machine entière fonctionne techniquement.

Niveau 5
Test d'Acceptation (UAT)

Le test final par le client. "Est-ce que j'ai ce que j'ai commandé ?" (Validation métier).
